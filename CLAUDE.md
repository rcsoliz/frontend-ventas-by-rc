## Design Context

Este proyecto tiene contexto de diseño capturado por la skill `impeccable`:

- **`PRODUCT.md`** — register `product` (backoffice de ventas), plataforma `web`, usuarios (vendedores/administradores), posicionamiento y principios de diseño.
- **`DESIGN.md`** — sistema visual real ("The Ledger"): paleta monocromática cálida, tipografía sin escala hero, casi sin sombra. Ver también el sidecar `.impeccable/design.json`.

Leer ambos antes de diseñar o modificar cualquier pantalla o componente de UI.
